# CYBER AYADI 💻

مرحبا بكم في فضاء **CYBER AYADI**
منصة بسيطة لعرض خدمات مقهى الإنترنت وطلبها بسهولة عبر واتساب.

---

## 🚀 فكرة المشروع

هذا الموقع عبارة عن صفحة هبوط (Landing Page) تحتوي على:

* عرض خدمات المقهى (الطباعة، الإنترنت، الخدمات الرقمية)
* متجر بسيط لشراء الخدمات
* نظام سلة (Cart)
* طلب مباشر عبر واتساب بدون تعقيد

---

## 🧩 الخدمات المقدمة

* 🖨️ الطباعة والنسخ (أبيض وأسود / ملون)
* 🌐 ساعات الإنترنت
* 📄 إعداد الوثائق (CV، طلبات…)
* 📷 Scan ونسخ الوثائق
* 💻 خدمات رقمية متنوعة

---

## 🛒 كيف يعمل الموقع؟

1. اختر المنتج أو الخدمة
2. اضغط "أضف إلى السلة"
3. عند الانتهاء اضغط "الدفع"
4. سيتم تحويلك إلى واتساب مع رسالة جاهزة تحتوي على طلبك
